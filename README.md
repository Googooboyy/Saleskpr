# SalesKpr

**Sell. Track. Grow.**

SalesKpr is a lightweight sales and product management dashboard for small print and stationery businesses. Track products, record sales, manage inventory, and monitor revenue—all in one place.

## Features

- **Product catalog** — Add and manage products (Notebooks, Bill Books, Vouchers, Business Cards, Flyers, and more)
- **Sales dashboard** — Total revenue, selected totals, and share-of-revenue metrics
- **Shopping cart** — Quick checkout flow for sales
- **Sales log** — View and manage sales history
- **Purchases** — Track incoming inventory
- **Product performance** — Rankings by sales and buyer activity
- **Organisation branding** — Custom name and logo for your business

## Quick Start

1. Open `index.html` in a web browser
2. No build step or server required—data is stored in your browser’s localStorage

## Tech Stack

- Plain HTML, CSS, and JavaScript
- LocalStorage for persistence
